Essere means 'to drink'. Bere is an irregular verb as it is different between tenses. The conjugation is also irregular between tenses.

| English     | Present    |
| ----------- | ---------- |
| I drink        | Io bevo    |
| You are     | Tu bevi     |
| You are     | Lei beve      |
| He/She      | Lui/Lei  beve |
| We are      | Noi beviamo  |
| You all are | Voi bevete  |
| They are    | Loro bevono  |
