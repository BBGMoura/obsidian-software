Words that end in -ere tend to follow the same pattern. This pattern can be seen below.  

| Subject | Verb Conjugation |
| ------- | ---------------- |
| io      | -o               |
| tu      | -i               |
| lui/lei | -e               |
| noi     | -iamo            |
| voi     | -ette            |
| loro    | -ono             | 

However, there can be an exception with irregular verbs.
