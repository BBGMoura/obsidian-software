Scrivere means 'to write'.

| English       | Present         |
| ------------- | --------------- |
| I write       | Io scrivo       |
| You write     | Tu scrivi       |
| You write     | Lei scrive      |
| He/She writes | Lui/Lei scrive |
| We write      | Noi scriviamo   |
| You all write | Voi scrivete    |
| They write    | Loro scrivono   |
